# Writeup

Du får et kryptert flagg, men kan også oppgi egen plaintekst for å få kryptert med samme nøkkel.

Hvis du så tar: `plaintekst` XOR `ciphertekst` vil du få krypteringsnøkkelen som flagget er kryptert med.
